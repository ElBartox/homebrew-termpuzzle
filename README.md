# homebrew-termpuzzle

Homebrew tap for [TermPuzzle](https://termpuzzle.com) — the workspace OS for AI builders.

## Install

```sh
brew install --cask ElBartox/termpuzzle/termpuzzle
```

Or tap first:

```sh
brew tap ElBartox/termpuzzle
brew install --cask termpuzzle
```

## Upgrade

TermPuzzle updates itself in-app. To bump the Homebrew record too:

```sh
brew upgrade --cask termpuzzle
```
